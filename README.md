Create a world-class, high-end landing page for "AXIS_ULTRA" (a private neobank). The design language is "Luxury Industrialism". Think of a physical object made of expensive raw materials (titanium, smoked glass, heavy concrete) translated into a digital interface.

1. COLOR PALETTE (The "Anti-Cheap" Rule):

Strict Monolith: Avoid pure blacks. Use Deep Graphite (#080808), Warm Gunmetal (#121212), and Raw Titanium (#E2E2E2) for typography.

Reflections: Instead of flat colors, use Anisotropic Metallic Reflections and Subtle Iridescence (like oil on dark water).

Accent: No neons. Use a single "Clinical White" or "Burned Orange" (#FF4D00) for 1px-thin accents only.

2. THE CENTRAL ARTIFACT (High-Complexity WebGL):

Object: A Protoplasmic Mercury Monolith (a shifting liquid metal slab).

Physics: It must possess High Density and Inertia. When the cursor moves, the monolith shouldn't just ripple; it should bulge and distort with heavy weight, like a magnet moving under a thick sheet of dark oil.

Surface: Macro-textures. Upon close-up (on scroll), show Micro-surface imperfections, brushed metal grain, and dust-like particles floating in 3D space.

3. ANIMATION LOGIC (Cinematic Motion):

Non-Linear Easing: No linear movements. Use "Spring Physics" (stiffness: 100, damping: 20). Everything must feel like it has physical mass.

Luma-Mask Transitions: Sections should reveal themselves through Light-Leak Masks or Optical Distortions (like looking through thick, heavy glass).

Parallax Depth: Implement 5 layers of depth. Far-background: Blurred technical schematics. Mid-ground: The Monolith. Foreground: Floating UI elements that react to mouse tilt (gyro-parallax).

4. TYPOGRAPHY & LAYOUT (Editorial High-Fashion):

Layout: An "Anti-Grid" system. Use massive white space (dead space) to create a feeling of exclusivity.

Fonts: * Display: A custom-looking, wide, "Stealth" Sans (e.g., Monument Extended or Neue Montreal), but with letter-spacing: -0.05em.

Detail: Use 7pt font size for technical specs, looking like engravings on a luxury watch.

Interactions: When hovering over text, the letters should "bleed" or "shimmer" like a glitching analog signal.

5. THE CUSTOM CURSOR (The "Scanner"):

Design: A razor-thin, 1px "Focus Ring" that acts as a Refraction Lens.

Effect: As the cursor passes over images or the Monolith, it should magnify and distort the content underneath (Glass Distortion Shader), revealing hidden layers of technical data.

6. TECHNICAL QUALITY TAGS:

Aesthetics: Stealth-Wealth, Brutalist Luxury, High-Density Materialism, Macro-Photography.

Effects: Ray-traced shadows, Anisotropic filtering, 120fps physics, Gaussian-blur glassmorphism (100px radius), Depth of Field (DoF).
